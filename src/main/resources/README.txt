PiplineDataBase(R) Version 1.0 19/02/2018

Перед использованием
---------------------

Пользователям:
- Потребуется установить Java. Можно использовать ссылку https://www.java.com/ru/download/windows-64bit.jsp
- Иметь локальный доступ к машине используемой как сервер.

Серверу:
- Установить MySQL Server, MySQL Workbench, MySQL Connector/J. Ссылка https://dev.mysql.com/downloads/
- Создать соединение в MySQL Workbench с логином: "root" и паролем: "root". В Hostname указать IP машины используемой как сервер.
- Если на сервере отсутствует база данных "nd_database" импортировать базу в MySQL Workbench -> Data Import -> Import from Self-Contained File (Выбираете сохраненный файл .sql, который содержит базу данных "nd_database" и импортируете).


Работа с программой:
---------------------

- Ввести логин и пароль подключения к базе (по умолчанию root/root).
- Работать.
- Бэкап базы данных желательно проводить каждый день. 


Возможные изменения:
---------------------

Вы можете поменять параметры прописанные в config.properties:
- Откройте файл при помощи блокнота.
- Файл содержит 3 параметра
  - URLofDB - URL базы данных; 
  - mysqldumpPath - путь, где лежит файл mysqldump.exe. Требуется для бэкапа базы данны;
  - pathToDump - путь, куда будет сохраняться .sql файл при бэкапе. ВНИМАНИЕ! В пути не должно быть пробелов. Бэкап базы данных желательно проводить каждый день. 
  - pathToArHive - путь к архиву. Если путь к архиву поменяется, внесите изменения.
- Сохраните файл в кодировке ANSI. ОБЯЗАТЕЛЬНО
